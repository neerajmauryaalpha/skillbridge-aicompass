# SkillBridge – AICompass 🤖

**SkillBridge: AICompass** is a conversational AI literacy web platform designed to help users understand Artificial Intelligence concepts, ethical risks, bias, transparency, and responsible AI usage through interactive dialogue and quizzes.

## 🎯 Project Objectives
- Explain AI concepts in simple language
- Educate users about AI bias, ethics, transparency, and risks
- Provide scenario-based and difficulty-based quizzes
- Demonstrate conversational UX and AI literacy design

## 🧠 Key Features
- Conversational chatbot interface
- Free-text AI question handling
- Scenario-based AI ethics quiz
- Difficulty-based learning paths (Beginner, Intermediate, Advanced)
- Accessible and responsive web UI

## 🛠 Tech Stack
- Frontend: React + Tailwind CSS
- UI Components: ShadCN UI
- State Management: React Hooks
- Design Focus: Conversational UX & Responsible AI
